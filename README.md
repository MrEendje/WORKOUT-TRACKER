# Workout Tracker (Local)

Minimal, mobile-first workout tracker using Tailwind CSS and LocalStorage.

Run locally:

1. Open `c:\PROJECTEN\GYMCALENDER\index.html` in your browser.

Notes:
- Uses Tailwind CDN for styling (no build step).
- Data is stored in browser LocalStorage under the key `workout-tracker:v1`.
- Designed mobile-first; inputs use numeric keyboards where appropriate.

If you want a build with Tailwind CLI, I can add a package.json and build pipeline.
